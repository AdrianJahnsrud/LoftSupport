## Vanlige spørsmål

**Hvordan logger jeg en økt?**
Trykk på "Start ny økt" på hjem-skjermen og velg øvelser.

**Hvordan ser jeg fremgangen min?**
Gå til "Aktivitet" i navigasjonsmenyen.

**Kan jeg slette data?**
Ja, gå til Profil og velg innstillinger.

## Om appen
Løft er en treningsapp for styrketrening. Last ned på App Store.
